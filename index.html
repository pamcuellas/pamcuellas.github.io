Latitude Analysis Dashboard

This is a dashboard website to present the result of an analysis of how weather changes as getting closer to the equator. We can check how the behaviour of humidity, cloudiness, temperature and wind speed are related to the latitude variation. A table with all data used for plotting the graphs is also available.

Technologies used in this project:
  • HTML5
  • CSS3
  • jQuery
  • Bootstrap
