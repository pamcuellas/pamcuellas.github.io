Mission to Mars

This is a scrape web application that visit some web sites with Mars subject to get information and pictures. On this page you can click on links to see the original source of that information.

Resources used to build this application:

  • Python
  • Flask
  • Splinter
  • flask_pymongo
  • JSON
  • HTML
  • Bootstrap
  • Fontawesome
  • MongoDB at MongoDB Atlas
  • And Heroku as hosting
